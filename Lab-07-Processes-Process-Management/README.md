# 🚀 Linux Fundamentals – Lab 07: Processes & Process Management

## 📌 Objective

The objective of this lab is to understand how Linux manages running programs (processes), how to monitor and control them, manage foreground and background jobs, adjust process priorities, and understand parent-child process relationships.

---

# 📖 What I Learned

In this lab, I learned:

- Difference between a Program and a Process
- Process Life Cycle
- Parent Process (PPID)
- Child Process
- Interactive Shell
- Foreground vs Background Processes
- Jobs vs Processes
- Process Signals
- Process Priority
- Process Tree
