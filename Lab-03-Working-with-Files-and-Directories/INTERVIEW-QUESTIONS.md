# Lab 03 Interview Questions

## Basic Questions

1. What is the purpose of the `touch` command?
2. What is the difference between a file and a directory?
3. What does the `mkdir` command do?
4. What is the purpose of the `-p` option in `mkdir`?
5. What is brace expansion in Bash?

## Intermediate Questions

6. What is the difference between `cp` and `mv`?
7. Why is `cp -r` required when copying directories?
8. What happens if you try to copy a directory without using `-r`?
9. What is the difference between `rm`, `rm -r`, and `rmdir`?
10. Why can't `rmdir` remove a non-empty directory?
11. What is the purpose of the `-i` option with `rm`, `cp`, and `mv`?
12. What does the `-v` option do?
13. What is the purpose of the `-f` option with `rm`?
14. What happens if the destination file already exists when using `cp` or `mv`?

## Practical Questions

15. How would you create five files named `log1.txt` to `log5.txt` using a single command?
16. How would you create nested directories in one command?
17. How would you copy an entire project directory while preserving the original?
18. How would you rename a file without changing its contents?
19. How would you move multiple files into another directory using a single command?
20. How would you safely delete a non-empty directory?

## Scenario-Based Questions

21. You need to back up a configuration file before editing it. Which command would you use?
22. You accidentally used `rm -rf` on an important directory. What are the possible recovery options?
23. A directory named `Projects` already exists. What happens if you run `mkdir Projects`?
24. Why did the command `touch log{1..3}.txt Logs` create files in the current directory instead of inside `Logs`?
25. What is the difference between changing into a directory with `cd` before creating files and specifying the directory path directly in the command?

These questions cover the core concepts of Linux file and directory management and are commonly asked in Linux administration, DevOps, and cybersecurity interviews.
