# Lab 03 Summary

In this lab, I learned how to manage files and directories using essential Linux commands. I created files with `touch`, organized data using `mkdir`, and built nested directory structures with the `-p` option. I practiced copying files and directories using `cp` and `cp -r`, moving and renaming them with `mv`, and safely removing files and directories using `rm`, `rm -r`, and `rmdir`.

I also explored useful command options such as interactive (`-i`), verbose (`-v`), and force (`-f`) modes. Additionally, I learned how shell brace expansion works for creating multiple files and directories efficiently and understood the importance of specifying directory paths correctly when using brace expansion.

Finally, I completed a real-world file management challenge that combined all the commands learned in this lab, strengthening my practical Linux administration skills.
